Wheels of Trucks is a simple truck name randomizer created in **2021**, inspired by Wheel of Names.  
You enter multiple truck names separated by commas, and the script highlights a random choice.

This project is preserved as an archive of early JavaScript experiments.

---

## 🚚 Features

- Enter unlimited truck names
- Automatic tag creation
- Random highlight animation
- Clean and minimal UI
- Fully client‑side (no backend)
